Description

This API provides functionality to get daily mess menus for a week. It supports querying for specific days (e.g., "Monday") or for the current or next day. The API also includes batch updates and retrieval of menus for a range of days within the week.

This API scrapes menu from https://www.ssms-pilani.in/ and stores it in json files for each day of the week. These json files can be updated (using /update or /updateall). Alternatively you can get the menu directly (/get or /getall) from the json without updating from the SSMS website, which is much faster.

get menu

This gets the menu from the json file of the given date and returns it.
To get the current day's menu you need to make a GET call to the following url :
https://messmenuapi.onrender.com/get


Query Parameters


This path accepts only a "day" query.
It can take the following values: tomorrow, monday, tuesday, ..., sunday.
If no query is passed, it defaults to the current day.


Example Usage


To fetch the next day's menu: https://messmenuapi.onrender.com/get?day=tomorrow
To get Saturday's mess menu: https://messmenuapi.onrender.com/get?day=saturday

Update menu

This first scrapes the menu from the SSMS website for the given day, stores it in the appropriate json file and then returns it. Getting the menu this way takes a significantly longer amount of time.

To update and get the current day's menu you need to make a GET call to the following url :
https://messmenuapi.onrender.com/update


Query Parameters and Usage


Query parameters are the same as /get.

To update and get the next day's menu: https://messmenuapi.onrender.com/get?day=tomorrow
To update and get Saturday's mess menu: https://messmenuapi.onrender.com/get?day=saturday

get and update multiple menus

These perform /get or /update actions for a range of days.
Response code is generated for each day in the given range, depending on the success of the operation.

To get the entire week's menu you need to make a GET call to the following url :
https://messmenuapi.onrender.com/getall

To update and fetch the entire week's menu you need to make a GET call to the following url :
https://messmenuapi.onrender.com/updateall


Query Parameters


These paths accept two queries: "start" and "end".
Both correspond to a day of week: monday, tuesday, wednesday, ..., sunday.

You can pass either one or both or none of these queries.
If not passed, start takes the default value of monday, and end takes the default value of sunday.
Both parameters are inclusive, i.e., getting the menus for the range: start = A and end = B would return the menus of A and B as well.


Example Usage


To get the menus between tuesday and friday, including both:
https://messmenuapi.onrender.com/getall?start=tuesday&end=friday

To update and get menus for saturday to wednesday, including both:
https://messmenuapi.onrender.com/updateall?start=saturday&end=wednesday

Response Codes

This API uses the following response codes:

Code Meaning
0 No error.
1 Unable to get/update menu. Usually due to the SSMS site not being updated.
2 Query parameter(s) passed are invalid.
